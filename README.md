# 📚 Study Buddy App

A simple and helpful app designed to improve your study sessions and keep you productive.

---

## 🚀 Features

* 🧠 **Flashcards**
  Quickly revise important concepts using interactive flashcards.

* ❓ **Quiz Mode**
  Test your knowledge with short quizzes.

* ⏳ **Pomodoro Timer**
  Stay focused using the Pomodoro technique (study + break cycles).

* 📝 **To-Do List**
  Organize your daily study tasks easily.

* 📊 **Progress Bar**
  Track your progress and stay motivated.

---

## 🎯 Purpose

This app helps students:

* Stay focused
* Revise efficiently
* Manage time better
* Track study progress

## 📌 How to Use

1. Add tasks in the To-Do list
2. Use Pomodoro timer to start studying
3. Revise using flashcards
4. Take quizzes to test yourself
5. Track your progress with the progress bar
